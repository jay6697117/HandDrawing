# 徒手画图大师 — 游戏设计规格文档

## 概述

一款移动端优先、兼容 PC 浏览器的徒手画图游戏。用户在 Canvas 上徒手画出指定的几何图形，系统通过几何特征算法计算用户所画图形与标准图形的相似度百分比，以此评分。

游戏融合**教育**与**休闲挑战**两种定位，面向**全年龄段**用户。通过关卡解锁系统，从简单图形到复杂组合图形渐进式挑战，兼具学习性和趣味性。

## 核心定位

- **类型**：教育 + 休闲挑战
- **用户**：全年龄段（儿童到成人）
- **平台**：移动端优先（Mobile First），兼容 PC 浏览器
- **风格**：可爱卡通风（鲜艳色彩、圆角、活泼有趣）

## 页面流程

```
🏠 首页 → 🗺️ 关卡地图 → 🎨 画图界面 → 📊 结果页
```

### 首页（HomeView）

- 游戏品牌标识和标题「徒手画图大师」
- 「开始游戏」主按钮（进入关卡地图）
- 「最高成绩」和「设置」辅助入口

### 关卡地图（LevelMapView）

- 按章节分组展示所有图形关卡
- 每个关卡卡片显示：图形图标、名称、历史最高相似度
- 未解锁章节显示锁定状态
- 解锁规则：完成前一章全部关卡（达到通关门槛）后解锁下一章

### 画图界面（DrawingView）

- 顶部显示目标图形名称和参考图
- 中间为 Canvas 画板区域（可选是否显示虚线参考）
- 底部操作按钮：「提交」和「清除」
- 画图时禁用页面滚动和缩放

### 结果页（ResultView）

- 对比展示：用户画的图形 vs 标准图形
- 相似度百分比（大字展示）
- 操作按钮：「再试一次」和「下一关」
- 达到通关门槛时显示庆祝动画

## 关卡系统

### 设计原则

**门槛随难度递减**——简单图形容易画好所以要求更高，复杂图形本身就难画所以门槛更低。保证每一章都"有挑战但不劝退"。

### 8 个章节，40 个图形

| 章节 | 主题 | 图形（5个/章） | 通关门槛 |
|------|------|---------------|---------|
| 第一章 | 基础圆与方 | 圆形、正方形、长方形、直线、椭圆 | 80% |
| 第二章 | 三角家族 | 等边三角形、直角三角形、等腰三角形、钝角三角形、锐角三角形 | 75% |
| 第三章 | 四边形进阶 | 菱形、梯形、平行四边形、直角梯形、风筝形 | 70% |
| 第四章 | 多边形挑战 | 五边形、六边形、七边形、八边形、十边形 | 65% |
| 第五章 | 星形与箭头 | 五角星、六角星、右箭头、上箭头、双向箭头 | 60% |
| 第六章 | 曲线艺术 | 爱心、月牙形、水滴形、云朵形、叶子形 | 55% |
| 第七章 | 符号与标志 | 十字形、闪电、无限符号、太极圆、螺旋 | 50% |
| 第八章（终极章） | 大师之路 | 房子形、猫脸、花朵、蝴蝶、圣诞树 | 45% |

## 图形识别算法

### 处理流程

1. **采集坐标点**：用户在 Canvas 上画图时，记录触摸/鼠标轨迹的所有坐标点
2. **预处理**：归一化到统一尺寸，消除位置和大小差异；平滑处理去除手抖噪点
3. **提取几何特征**：分析顶点数量、内角角度、边长比例、曲率变化、是否闭合、对称性等
4. **计算相似度**：将用户图形与目标标准图形的关键特征逐项对比，加权计算最终相似度百分比

### 各类图形的识别重点

| 图形类型 | 识别关键特征 |
|---------|------------|
| 圆形/椭圆 | 曲率恒定性、闭合度、长短轴比 |
| 三角形 | 顶点数=3、角度之和≈180°、边长比例 |
| 四边形 | 顶点数=4、对边平行性、角度、边长比 |
| 多边形 | 顶点数、边长均匀性、内角均匀性 |
| 星形 | 凹凸顶点交替、角度分布均匀性 |
| 曲线图形 | 关键点位置、曲率变化模式、对称性 |

## 评分机制

- **评分方式**：相似度百分比（0% - 100%）
- **展示方式**：大字百分比 + 对比图（用户画的 vs 标准图形）
- **记录方式**：每个关卡保存历史最高相似度
- **通关条件**：相似度 ≥ 当前章节通关门槛

## 移动端适配策略

- **触摸事件优先**：`touchstart`/`touchmove`/`touchend` 优先，`pointer` 事件作为后备兼容 PC
- **Canvas 自适应**：宽度 100%（移动端），PC 端限制 `max-width: 480px` 居中展示
- **防误触**：画图区域激活时，禁用页面滚动（`touch-action: none`）和双指缩放
- **响应式 UI**：所有按钮和文字适配移动端尺寸，点击区域 ≥ 44px
